Sitemap

Published:2025-01-01
  • Print
  • Forward

For Internet Explorer (IE), press Alt + the assigned letter to use an access key. For Firefox, press Alt + Shift + the assigned letter. The access keys (accessKey) on this website are set as follows:

  • Alt+U Top functional block, including Homepage, Sitemap, Site Search, Font Size Selection, Version Selection, etc.
  • Alt+C Central content block, the main content area of this page.
  • Alt+Z Bottom functional block.
  • Alt+S Site search.
  • If you are using Firefox, the method for using access keys is Shift+Alt+(access key letter). For example, pressing Shift+Alt+C will jump to the central content area of the page, and so on.

※ When tabs on this website cannot be selected with a mouse, you can use the following keyboard commands to navigate the content:
← → or ↑↓: Press the left/right or up/down arrow keys to move between tabs.
Home or End→: Jumps directly to the first or last tab.
Tab: After focusing on a tab, press the Tab key to jump to its content. When encountering radio buttons, please use the ← → or ↑↓ keys to navigate between the options.
Tab + Shift: Press Tab + Shift to jump back to the previous item. When focus returns to the tabs, you can continue to use the ← → or ↑↓ keys to navigate between them.
